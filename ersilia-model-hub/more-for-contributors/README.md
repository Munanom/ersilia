## Versioning

Ersilia uses[ semantic versioning](https://semver.org/), i.e. `{major}.{minor}.{patch}`. GitHub tags and Ersilia versions are synchronized. This means that Ersilia 0.1.14 corresponds to the GitHub tag v0.1.14.
